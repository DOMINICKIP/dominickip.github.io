Poultry Disease Detection Using Deep Learning

This notebook demonstrates a convolutional neural network (CNN) model for detecting poultry diseases based on image classification. The model is trained to distinguish between healthy chickens and those affected by diseases like Newcastle Disease (NCD), Coccidiosis, and Salmonella.

Using TensorFlow and Keras, the notebook preprocesses image data, applies data augmentation, constructs a CNN architecture, and evaluates its performance. This is particularly useful for early disease detection in poultry farms to reduce mortality rates.
