IoT Monitoring System

The IoT Monitoring System is developed using a combination of embedded systems and cloud technologies. It provides real-time monitoring of sensors and devices through a centralized dashboard, allowing users to track and analyze environmental data such as temperature, humidity, and device status.

The system works by connecting IoT devices to a cloud-based platform, with real-time data being sent and displayed on the dashboard for easy monitoring and control. The project utilizes technologies such as:

  • MQTT Protocol: For lightweight communication between IoT devices and the server.
  • Node.js: Backend API to manage device data and interactions.
  • React: Frontend dashboard to visualize data and monitor devices.
  • AWS IoT Core: For managing and scaling IoT devices in the cloud.

Here is the code for the IoT Monitoring System:


# Node.js Backend Example

const mqtt = require('mqtt');
const express = require('express');
const app = express();
const client = mqtt.connect('mqtt://your-mqtt-broker-url');

// Subscribe to topic
client.on('connect', function () {
    client.subscribe('iot/sensor/data', function (err) {
        if (err) {
            console.log('Error subscribing:', err);
        }
    });
});

// Handle incoming sensor data
client.on('message', function (topic, message) {
    // Process incoming message (sensor data)
    const sensorData = JSON.parse(message.toString());
    console.log('Received data:', sensorData);
});

// Express API to fetch sensor data
app.get('/api/sensors', (req, res) => {
    res.json({ temperature: 24, humidity: 60 }); // Example data
});

app.listen(3000, () => {
    console.log('IoT Monitoring System API running on port 3000');
});
